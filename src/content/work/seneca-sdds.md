## Overview

Developed and maintained the official website for Seneca College's School of Software Design and Data Science. The website serves as the primary information hub for students, faculty, and prospective applicants, providing comprehensive details about academic programs, faculty resources, and institutional guidelines.

**Note**: This website can only be accessed using Seneca College's VPN for security and privacy reasons.

## Key Features

### Academic Program Information
- Comprehensive program listings and course descriptions
- Curriculum details and learning outcomes
- Admission requirements and application processes
- Student resources and academic support information

### Faculty Resources
- Internal documentation for faculty members
- Compliance guidelines and academic policies
- Course management tools and resources
- Faculty collaboration and communication features

### Content Management
- WordPress-powered CMS for easy content updates
- Role-based access control for different user types
- Responsive design for all device types
- SEO-optimized content structure

## Technical Implementation

- **Content Management**: WordPress with custom themes and plugins
- **Frontend Framework**: Bootstrap 5 for responsive design
- **Database**: MySQL for content and user data storage
- **Technologies**: HTML5, CSS3, JavaScript for interactive features
- **Security**: VPN-restricted access for institutional security
