## Overview

Built and maintained the official website for Seneca College's School of Information Technology Administration & Security. The platform provides essential information about IT programs, cybersecurity courses, and serves as a resource hub for both students and faculty members.

**Note**: Access restricted to Seneca College VPN for institutional security.

## Key Features

### Program Showcase
- IT Administration program details
- Cybersecurity and network security courses
- System administration and infrastructure programs
- Career pathways and industry certifications

### Faculty Portal
- Internal resources for IT faculty
- Compliance and accreditation materials
- Course planning and curriculum development tools
- Student assessment and grading resources

### Student Resources
- Lab schedules and technical requirements
- Software and hardware specifications
- Industry partnerships and co-op opportunities
- Alumni success stories and career outcomes

## Technical Stack

- **Platform**: WordPress CMS with custom development
- **UI Framework**: Bootstrap 5 for mobile-first responsive design
- **Database**: MySQL for robust data management
- **Frontend**: HTML5, CSS3, JavaScript for enhanced interactivity
- **Access Control**: VPN-based authentication for secure access
