## Overview

Magnet is a comprehensive job portal and professional development platform that I contributed to during my time as a web developer at WhoPlusYou. The platform combines job search functionality with community features and integrated learning management systems to create a holistic career advancement ecosystem.

## Key Features

### Job Portal
- Advanced job search with filtering by industry, location, and experience level
- Job recommendations based on user profiles and preferences
- Application tracking and status management
- Employer profiles and company reviews
- Salary insights and market data

### Community Platform
- Professional networking and connection building
- Industry-specific discussion forums and groups
- Mentorship matching programs
- Event listings for networking and professional development
- User-generated content and knowledge sharing

### Learning Management System (LMS)
- Integrated course catalog for skill development
- Interactive video lessons and tutorials
- Quizzes and assessments for skill validation
- Progress tracking and certification programs
- Industry-recognized credentials and badges

## Technical Stack

### Backend
- **Framework**: Laravel (PHP) for robust MVC architecture
- **Database**: MySQL for relational data management
- **Authentication**: Laravel Sanctum for API authentication
- **Queue System**: Laravel queues for background job processing

### Frontend
- **Framework**: Vue.js for reactive single-page application
- **Language**: TypeScript for type-safe JavaScript development
- **Styling**: Tailwind CSS and Bootstrap 5 for responsive design
- **Legacy Support**: jQuery for backwards compatibility with older components

### Development & Architecture
- **API Design**: RESTful API architecture
- **State Management**: Vuex for centralized state management
- **Build Tool**: Webpack for asset compilation
- **Code Quality**: ESLint and Prettier for consistent code style

## My Contributions

As part of the development team at WhoPlusYou, I worked on:
