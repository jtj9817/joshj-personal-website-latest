## Overview

Developed a comprehensive web application for students enrolled in Seneca College's Fire Protection Engineering Technology program. The application streamlines the enrollment process, allowing students to select courses, manage their schedules, and complete payments through integrated online payment systems.

## Key Features

### Course Enrollment System
- Browse available Fire Protection courses by semester
- View detailed course descriptions and prerequisites
- Real-time seat availability and waitlist management
- Schedule conflict detection and resolution

### Payment Processing
- Secure online payment integration
- Multiple payment method support
- Automated invoice generation
- Payment history and receipt management

### Student Portal
- Personalized dashboard with enrolled courses
- Academic calendar and important dates
- Course materials and resource downloads
- Grades and progress tracking

### Administrative Tools
- Course catalog management
- Student enrollment reporting
- Payment reconciliation and tracking
- User access control and permissions

## Technical Architecture

- **Backend**: Node.js for scalable server-side processing
- **Frontend Framework**: Angular for dynamic single-page application
- **Database**: Microsoft SQL Server for transactional data integrity
- **UI Library**: Material UI for consistent component design
- **Frontend Technologies**: HTML5, CSS3, JavaScript/TypeScript
- **Architecture**: RESTful API design with JWT authentication

## Implementation Highlights

The application features a modern, responsive interface built with Angular and Material UI components. The Node.js backend handles complex business logic for course enrollment rules, payment processing, and data validation. Microsoft SQL Server ensures ACID compliance for critical enrollment and payment transactions.
